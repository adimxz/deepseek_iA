# 🤖 DeepSeek-Terminal

**Terminalde çalışan, çok dilli, yapay zeka destekli asistan**

> "Sana yardım etmek için buradayım reis!"

## ✨ Özellikler

- 🗣️ **7 dil desteği** (Türkçe, English, Deutsch, Français, Русский, العربية, 日本語)
- 🧠 **Hafıza** - Konuşmaları hatırlar
- 😊 **Duygu analizi** - Üzüldüğünü anlar, şaka yapar
- 💻 **Komut çalıştırma** - Terminal komutlarını güvenle çalıştırır
- 🔍 **Otomatik dil algılama** - Hangi dili konuştuğunu anlar
- 📝 **Konuşma geçmişi** - Önceki konuşmaları kaydeder
